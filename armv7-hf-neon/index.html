This is a test build collection of libretro cores, for armv7-hf-neon platform.
It is suitable for machines with Linux OS, architecture ARMv7, 32-bit. CPU should support hard-float and NEON (Raspberry Pi 2 as an example).

To download cores from here, point buildbot_core_download_url to here, either setting it from menu or editing your retroarch.cfg:
core_updater_buildbot_cores_url = "https://zoltanvb.github.io/armv7-hf-neon/"
Then use online updater / core downloader.

Note: This is an unofficial repo, created as a test for a cross-compilation environment. Keep following things in mind:
  • Cores are compiled with "unix-armv7-hardfloat-neon" target. It may result in lower performance than a build targeted to the specific HW platform such as "rpi2".
  • All possible cores are compiled, without consideration. So several of them may be unusably slow.
  • There may be compilation errors as well. If you find a core that is not working at all, and you're willing to do some testing, please open an issue here.
