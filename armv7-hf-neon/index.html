This is a test build collection of libretro cores, for armv7-hf-neon platform. It is suitable for machines with Linux OS, architecture ARMv7, 32-bit. CPU should support hard-float and NEON (Raspberry Pi 2 as an example). To download cores from here, edit your retroarch.cfg, and point buildbot_core_download_url to here: core_updater_buildbot_cores_url = "https://zoltanvb.github.io/armv7-hf-neon/" Then use online updater / core downloader.
